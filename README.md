## Getting started

- Get [miniconda](https://www.anaconda.com/docs/getting-started/miniconda/main)
- Initialize your environment: `conda create --file environment.yml`
- Activate the environment: `conda activate eradiate-training-20251016`
- Check your Eradiate installation: `eradiate sys-info`
- Download the support data: `eradiate data install core`
- Launch a Jupyter Lab session: `jupyter lab`
